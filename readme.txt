WP woocommerce checkout form display in popup 

== Description ==

This free Wordpress plugin allows to display checkout form in popup 

This plugin automatically overwrite checkout functionality in popup form

== Installation ==

* Upload the directory '/popup-checkout-plugin/' to your WP plugins directory and activate from the Dashboard of the main blog.
* When plugin is activated then automatically checkout popup/modal is done, when we click on "Proceed to checkout" button then automatically checkout popup is opened.

* This plugin is purely customization user can customize checkout popup according to your site design.
